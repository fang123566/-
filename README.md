# 解码徽派民居 | 宏村·空间密码与文化传承

徽派建筑文化信息交互可视化展示网站。

## 技术栈

- **HTML5** — 语义化页面结构
- **CSS3** — 纯 CSS 样式、动画、响应式布局
- **JavaScript (ES6+)** — 原生 JS 交互逻辑
- **ECharts 5** — 数据可视化图表库 (CDN 引入)

## 功能模块

- 首页英雄区 + 背景视频 + 云粒子动效
- 交互式全景探索（热点悬浮提示）
- 村落信息卡片（展开/折叠）
- 6 组 ECharts 数据可视化（建筑类型、年代分布、文化价值雷达图、三雕统计、历史时间轴、游客趋势）
- 遗产价值体系展示
- 滚动进入动画 + 导航自动高亮

## 本地运行

无需安装依赖，直接使用任意静态服务器即可：

```bash
npx http-server . -p 3001 -c-1
```

然后访问 `http://localhost:3001`

或直接在浏览器中打开 `index.html`（部分功能如视频可能受浏览器本地文件安全策略限制）。
